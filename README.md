# Marmi Meldola — Materiali promozionali

Repository contenente versioni HTML della brochure e della locandina di Marmi Meldola, pronte per stampa o esportazione in PDF.

File principali
- [Brochure.html](Brochure.html) — Brochure formattata per stampa A4 (usa Tailwind CSS).
- [Locandina.html](Locandina.html) — Locandina/poster formattata per stampa A5.
- [Marmi Meldola - Locandina.pdf](Marmi%20Meldola%20-%20Locandina.pdf) — PDF della locandina.
- `upload/` — cartella risorse (logo, immagini usate nelle pagine).

Istruzioni rapide per esportare in PDF
1. Apri il file HTML nel browser (consigliato: Chrome o Edge).
2. Apri Stampa (Ctrl+P) e seleziona "Salva come PDF" come destinazione.
3. Nelle impostazioni di stampa:
	 - Imposta formato carta su A4 (per `Brochure.html`) o A5 (per `Locandina.html`).
	 - Abilita "Stampa sfondi/graphics di background".
	 - Imposta i margini in base alla regola `@page` (A4: 10mm) o lascia "Default".
	 - Scala 100% (se necessario regolare per adattare).
4. Salva il PDF e verifica che tutte le immagini siano incorporate.
